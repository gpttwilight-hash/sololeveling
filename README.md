# Система Пробуждения (Solo Leveling)

«Система Пробуждения» — это современное PWA-приложение для геймификации личного развития, созданное по мотивам популярной вселенной **Solo Leveling**. Приложение превращает повседневные привычки, задачи и цели в RPG-систему с прокачкой персонажа, ранговой системой и квестами.

## 🌟 Особенности проекта

В отличие от классических трекеров привычек, «Система Пробуждения» использует научно обоснованные механики поведенческого дизайна, эстетику Dark Premium и анти-грайнд механики, защищающие пользователя от выгорания.

### 🔥 Основной функционал

- **Система Атрибутов (Характеристики):**
  Пользователь прокачивает свои параметры в реальной жизни: 
  - **STR (Сила):** Физическое здоровье и спорт.
  - **INT (Интеллект):** Знания, обучение, навыки.
  - **CHA (Харизма):** Коммуникация и социальное влияние.
  - **DIS (Дисциплина):** Сила воли и самоконтроль.
  - **WLT (Богатство):** Финансы и карьера.

- **Многоуровневая система квестов:**
  - *Ежедневные задания (Daily)* для поддержания рутины.
  - *Еженедельные задания (Weekly)* на краткосрочные цели.
  - *Эпические квесты (Boss Raids)* для больших жизненных целей со шкалой прогресса и подзадачами.

- **Уровни и Ранги:**
  - Набор опыта (XP) за выполнение квестов.
  - Развитие от E-ранга (Пробуждённый) до S-ранга (Монарх) через прохождение специальных испытаний-рангапитаний (Trials).

- **Анти-грайнд механики (Забота о пользователе):**
  - Поддержка Rest Days (дней отдыха) без потери стрика (streak).
  - Умная система дебаффов вместо жестких наказаний и сброса прогресса.
  - Адаптивная сложность на основе регулярности выполнения заданий.

- **Магазин наград:**
  - Заработок виртуальных монет за достижения.
  - Возможность тратить монеты на реальные поощрения, настроенные самим пользователем (например, «Покупка для хобби», «Заказать любимую еду» и т.п.).

## 💻 Технический стек

**Фронтенд:**
- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4 + shadcn/ui
- Zustand (Управление стейтом)
- React Hook Form + Zod

**Бэкенд & Инфраструктура:**
- Supabase (PostgreSQL, Auth, Edge Functions)
- Prisma ORM
- Vercel (Хостинг)
- next-pwa (Офлайн-поддержка)

## 🚀 Запуск проекта для разработки

1. Перейдите в папку с основным приложением:
   ```bash
   cd awakening-system
   ```
2. Установите зависимости (если еще не установлены):
   ```bash
   npm install
   ```
3. Настройте файл `.env.local` на основе ваших ключей Supabase и других доступов (вы можете скопировать `.env.example`).
4. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
5. Откройте [http://localhost:3000](http://localhost:3000) в браузере.
